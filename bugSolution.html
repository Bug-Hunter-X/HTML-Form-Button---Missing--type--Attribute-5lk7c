The solution is to add the `type` attribute to the button element within the form. This clarifies the button's intended behavior to the browser.

<form id="myForm" action="/submit">
  <button type="submit" onclick="alert('Button clicked!')">Submit</button>
</form>

By specifying `type="submit"`, the button now correctly submits the form. Other common values for the type attribute include `button` (default behavior), `reset` (resets the form), etc.
